# 窗口2：数学（北师大版）知识资料收集

## 任务概述
为中考知识库系统收集**北师大版数学**学科的完整学习资料，覆盖初中三年全部内容。

## 学科详情
- **年级**：七年级上/下、八年级上/下、九年级上/下
- **教材版本**：北京师范大学出版社
- **分册**：七上、七下、八上、八下、九上、九下

---

## 资料收集要求（必须包含以下全部类型）

### 1. 课本本体内容
- 各章节教材原文定义、定理
- 教材章节结构与目录
- 公式推导过程
- 章节导入问题

### 2. 知识点归纳
- 代数模块（数与式、方程与不等式、函数）
- 几何模块（三角形、四边形、圆、相似、锐角三角函数）
- 统计与概率模块
- 核心公式速记表
- 定理证明汇总

### 3. 例题与习题
- 教材经典例题（带详细解答）
- 课后习题精选（含答案）
- 教材"想一想""做一做"题目分类

### 4. 答题套路与技巧
- 选择题秒杀技巧
- 填空题快速解题方法
- 解答题规范书写格式
- 压轴题解题思路模板

### 5. 万维/必刷题类似题
- 中考数学真题分类汇编
- 各地模拟题精选
- 几何辅助线添加技巧题库
- 函数图像变换专题

### 6. 课本原题
- 近年中考数学原题（北师大版改编题）
- 教材母题变式题
- 中考几何证明原题

### 7. 扩展内容
- 初中数学公式大全（可打印版）
- 数学思想方法总结（分类讨论、数形结合等）
- 竞赛入门题精选
- 数学易错点警示录

### 8. 中考复习专题
- 中考数学冲刺专题（按题型）
- 中考模拟卷（含详细答案）
- 中考解题时间分配策略
- 压轴题专项突破（函数、几何）

### 9. 中考高频考点
- 数与式：实数运算、整式因式分解、分式化简
- 方程与不等式：一元二次方程、应用题
- 函数：一次函数、反比例函数、二次函数
- 几何：全等三角形、相似三角形、圆的综合
- 统计与概率：数据分析、概率计算

---

## 输出格式要求

**对于每个知识点，请提供：**
```json
{
  "title": "知识点名称",
  "chapter": "所属章节",
  "category": ["代数", "几何", "统计"],
  "difficulty": "基础|中等|困难",
  "formula": "相关公式（如果有）",
  "content": "详细解释",
  "examples": ["例题列表"],
  "tips": "解题技巧",
  "tags": ["高频考点", "中考必考"]
}
```

**输出保存位置**：
```
/知识库/src/data/subjects/math/
```

## 资料收集批次

### 第一批（基础框架）
1. 创建学科目录结构
2. 收集课本章节大纲
3. 整理核心公式定理清单

### 第二批（例题详解）
1. 各章节经典例题录入
2. 课后习题答案整理
3. 答题模板编写

### 第三批（真题与扩展）
1. 中考真题分类汇编
2. 万维必刷题收录
3. 压轴题专题突破

## 注意事项
- 公式必须**LaTeX格式**保存
- 几何题需要**图形描述**或配图说明
- 函数题需要**图像分析**
- 所有题目都要有**详细解答过程**
