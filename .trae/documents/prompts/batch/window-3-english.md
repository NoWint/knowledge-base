# 窗口3：英语（沪教版）知识资料收集

## 任务概述
为中考知识库系统收集**沪教版英语**学科的完整学习资料。

## 学科详情
- **年级**：七年级上/下、八年级上/下、九年级上/下
- **教材版本**：上海教育出版社（沪教版/牛津版）
- **特色**：模块化教学、话题整合

---

## 资料收集要求

### 1. 课本本体内容
- 各单元话题与功能意念
- 单元核心词汇表（含音标）
- 单元语法聚焦
- Reading原文与翻译
- 教材目录与模块结构

### 2. 知识点归纳
- 初中英语语法体系完整梳理
- 词法：名词、动词、形容词、副词、代词、介词、连词
- 句法：五大基本句型、宾语从句、定语从句、状语从句
- 时态：一般现在/过去/将来、现在/过去/将来进行时、现在/过去完成时
- 被动语态
- 虚拟语气基础

### 3. 词汇表
- 中考核心词汇1600词
- 不规则动词表
- 词性转换汇总
- 短语动词分类
- 固定搭配与习惯用语

### 4. 例题与习题
- 教材Reading理解题目
- Grammar练习题
- 单元测试题精选
- 中考语法单项选择题分类

### 5. 答题套路与技巧
- 阅读理解题型分类与技巧
- 完形填空解题方法
- 任务型阅读答题技巧
- 写作万能句型与模板
- 听力场景词汇汇总

### 6. 万维/必刷题类似题
- 中考英语真题分类
- 各地完形填空真题
- 阅读理解题型分类
- 写作话题预测与范文

### 7. 课本原题
- 中考英语原题（上海卷为主）
- 教材改编的中考题
- 课本句子在真题中的再现

### 8. 中考专题复习
- 词汇运用专题
- 语法选择专题
- 完形填空专题
- 阅读理解专题（记叙文/说明文/议论文）
- 写作提升专题
- 中考模拟卷（含答案）

### 9. 沪教版特色
- Module 1-6 各模块核心内容
- Project板块活动资料
- Culture Corner文化拓展
- 课本配套音频场景词汇

---

## 输出格式要求

**词汇条目格式：**
```json
{
  "word": "单词",
  "phonetic": "/音标/",
  "translation": "中文释义",
  "partOfSpeech": "词性",
  "examples": ["例句"],
  "phrases": ["短语搭配"],
  "derivatives": ["派生词"],
  "tags": ["中考核心", "常考"]
}
```

**语法条目格式：**
```json
{
  "title": "语法名称",
  "structure": "基本结构",
  "usage": "用法说明",
  "examples": [
    {
      "sentence": "例句",
      "translation": "翻译",
      "analysis": "分析"
    }
  ],
  "tips": "使用技巧",
  "commonMistakes": ["常见错误"]
}
```

**输出保存位置**：
```
/知识库/src/data/subjects/english/
```

2. 模拟卷与冲刺卷

- 作文必须有**多种范文**（基础/进阶）
- 听力场景词汇要按**主题分类**
