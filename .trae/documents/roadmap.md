# Eric 知识库 - 未来大方向规划

> **规划日期**: 2026-04-24
> **当前阶段**: MVP 完成，正进行第二轮功能扩展

---

## 一、项目愿景

**打造一个本地优先的中考备考知识管理系统，让知识像星座一样可视化，让复习像游戏一样上瘾。**

## 三、三个核心价值方向

### 1. 🎯 知识图谱 + 主动回忆闭环
**差异化**: Anki 无图谱，Obsidian 无练习，我们两者结合

### 2. 🔒 本地优先，数据主权
**差异化**: Quizlet/RemNote 必须上云，我们纯本地

### 3. 🎓 中考备考垂直场景
**差异化**: 通用工具需要大量配置，我们开箱即用

---

### Phase 2: 学习体验升级

**目标**: 让学习变得更有趣、更有效

#### 2.1 游戏化元素
- [ ] **连续学习天数** -  streaks 系统
- [ ] **学习成就徽章** - 解锁成就激励
- [ ] **知识点掌握等级** - 从青铜到王者的晋升
- [ ] **每日挑战** - 随机难题奖励

#### 2.2 复习提醒系统
- [ ] **浏览器通知** - 到点提醒复习
- [ ] **Service Worker** - 离线推送
- [ ] **复习日历** - 直观看到哪天该复习

#### 2.3 智能推荐
- [ ] **薄弱点优先** - 自动推荐最该复习的内容
- [ ] **艾宾浩斯曲线** - 科学安排复习间隔
- [ ] **知识点关联推荐** - 学了这个，推荐那个

---

### Phase 3: 内容生态

**目标**: 降低内容生产成本

#### 3.1 内容导入
- [ ] **更多格式支持** - JSON, CSV, Markdown
- [ ] **批量导入优化** - 模板下载、格式验证
- [ ] **从图片 OCR** - 拍照导入题目（需 AI）

#### 3.2 内容生成（可选 AI）
- [ ] **AI 生成变体题** - 同一知识点生成不同题目
- [ ] **AI 生成闪卡** - 自动从知识点生成问答
- [ ] **AI 错因分析** - 分析错误原因给出建议

#### 3.3 资源共享
- [ ] **题库分享** - 导入/导出题库包
- [ ] **知识图谱模板** - 预置学科图谱

---

### Phase 4: 多端同步

**目标**: 打破设备限制

#### 4.1 本地同步方案
- [ ] **IndexedDB → 文件导出** - 完整数据备份
- [ ] **增量同步** - 只同步变化的部分
- [ ] **冲突解决** - 多设备修改时的合并策略

#### 4.2 云端同步（可选）
- [ ] **Supabase 集成** - 免费 BaaS
- [ ] **端到端加密** - 数据加密后存储
- [ ] **实时同步** - 多设备实时更新

---

### Phase 5: 平台化

**目标**: 从个人工具到学习社区

#### 5.1 多用户管理
- [ ] **家长监管** - 查看孩子学习情况
- [ ] **学习小组** - 好友之间分享进度
- [ ] **竞争排行** - 学习时长/正确率排行

#### 5.2 数据分析增强
- [ ] **详细周报/月报** - 生成 PDF 报告
- [ ] **能力雷达图** - 可视化各科能力
- [ ] **进度预测** - 预计何时达到目标

#### 5.3 考试功能增强
- [ ] **真题库** - 历年真题导入
- [ ] **模拟考试报告** - 详细分析报告
- [ ] **错题归类** - 按知识点/错误原因归类
